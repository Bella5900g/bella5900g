## 🚀 Como Hospedar no GitHub Pages

### **Passo 1: Criar Repositório**
1. Acesse [GitHub](https://github.com)
2. Clique em "New repository"
3. Nome: `isabella-vieira-portfolio` (ou seu nome de usuário)
4. Marque "Public"
5. Clique em "Create repository"

### **Passo 2: Upload dos Arquivos**
1. Faça upload de todos os arquivos do portfolio
2. Ou use o GitHub Desktop para clonar e sincronizar

### **Passo 3: Configurar GitHub Pages**
1. Vá em **Settings** do repositório
2. Role até **Pages** no menu lateral
3. Em **Source**, selecione "Deploy from a branch"
4. Escolha "main" branch
5. Clique em **Save**

### **Passo 4: Acessar o Portfolio**
- URL será: `https://seu-usuario.github.io/isabella-vieira-portfolio`
- Aguarde alguns minutos para o deploy
